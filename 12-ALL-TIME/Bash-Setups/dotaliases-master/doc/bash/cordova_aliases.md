### Cordova build ###
- **coba**: `cordova build android`
- **cobi**: `cordova build ios`

### Cordova emulate ###
- **coea**: `cordova emulate android`
- **coei**: `cordova emulate ios`

### Cordova run ###
- **cora**: `cordova run android --device`
- **cori**: `cordova run ios --device`
