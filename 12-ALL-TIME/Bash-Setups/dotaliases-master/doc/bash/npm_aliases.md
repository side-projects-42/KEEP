### Npm ###
- **np**: `npm`

### Npm install ###
- **npi**: `npm install`
- **npis**: `npm install --save`
- **npig**: `npm install -g`

### Npm update ###
- **npu**: `npm update`
- **npug**: `npm update -g`

### Npm search ###
- **nps**: `npm search`
